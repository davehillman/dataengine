About

This application illustrates a data model and demo capability.

The demo capabilities include:

1. Data Viewer: basic functionality for a data model based on simulating key:value using a relation database system.

2. Data Commenter: Variation focused on a user capability to comment as desired.

3. Graphs: Examples of graphing tools.

4. Files: Examples of acessing and displaying file-based information.

5. Maps: Using markers and graphics to annotate maps (Google Maps). Please note: this appliation will open in a new tab due to the way that Google Maps interacts with the browser.
